# EN
## WT Simple price table Joomla module
WT Simple Price Table is a Joomla module for quickly creating price lists in structured tables with product or service
groups. Within a single module instance, you can add multiple sections, set item titles, images, current and old prices,
currency, and an optional “from” prefix. It supports multiple output layouts, including UIkit-based table and accordion views,
making it easy to integrate pricing blocks into any site template.

# RU
## WT Simple price table - модуль прайслиста для Joomla
WT Simple Price Table — модуль Joomla для быстрого создания прайс-листов в виде удобных таблиц с группами товаров или услуг. Внутри одного модуля можно добавить несколько секций, задать название позиции, изображение, текущую и старую цену, валюту и префикс «от». Поддерживает несколько макетов вывода, включая UIkit-варианты (таблица и аккордеон), что позволяет легко встроить прайс в любой шаблон сайта.
